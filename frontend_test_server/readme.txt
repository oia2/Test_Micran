Инструкция

1.Установить python версии 3.10.12 (можно выше, но на этой версии точно работает)

2. После установки python нудно установить все библиотеки с помощью команды
pip install -r requirements.txt

3. Для запуска тестового сервера используем команду
python __main__.py

4. Подключение производится по адресу 0.0.0.0:8000
